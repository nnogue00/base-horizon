# Base Horizon (BHZN)

Base Horizon is a Web3 learning project built on Base.

## Mission

To learn blockchain development, smart contracts and decentralized applications while building in public.

## Token

- Name: Base Horizon
- Symbol: BHZN
- Network: Base

## Goals

- Learn Solidity
- Deploy smart contracts
- Build decentralized applications
- Contribute to Base ecosystem
